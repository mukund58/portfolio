
# Portfolio

A personal portfolio website built with [Zola](https://www.getzola.org/).

## Features

- Responsive design
- Section for projects and experience
- Easy to customize

## Tech Stack

- HTML
- CSS
- JavaScript
- TeX

## Getting Started

To run locally:

```bash
git clone https://github.com/mukund58/portfolio.git
cd portfolio
zola serve
```

Then visit `http://localhost:1111` in your browser.
